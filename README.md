# Terzina_data_ana
This repository contains the scripts to analyse the binary data and csv from Terzina DAQ data taking


📌 Contribution Workflow

For any change beyond the initial commit:

    Create an Issue
    Create a branch named after the issue
    Work on that branch
    Assign assignees and reviewers
    Open a Merge/Pull Request when ready

❗ No direct commits to main. All changes must go through issue → branch → PR/MR.
